# Getting Started - Finance Hub

## Início Rápido

### 1. Acesse o App
https://app.finance-hub.com

### 2. Crie sua Conta
- Email
- Senha segura (min 8 caracteres)
- Confirme email

### 3. Configuração Inicial

#### Dados da Empresa
- CNPJ
- Razão Social
- Endereço

#### Integrações (Opcional)
- Bitrix24
- Banco (Open Finance)
- WhatsApp Business

### 4. Primeiros Passos

**Cadastre um fornecedor:**
Menu → Fornecedores → Novo

**Crie uma conta a pagar:**
Menu → Contas a Pagar → Nova Conta

**Veja o dashboard:**
Dashboard mostra visão geral financeira
